# 🌍 TravelSphere — DevOps Capstone Project

TravelSphere is a full-stack travel booking platform built as a DevOps capstone project, demonstrating end-to-end CI/CD pipelines, containerisation, and cloud-native deployment.

---

## 📁 Project Structure

```
TravelSphere-DevOps-Project/
├── backend/          # Java Spring Boot REST API
├── frontend/         # React + Tailwind CSS UI
├── kubernetes/       # Kubernetes YAML manifests (Deployments, Services, Ingress)
├── jenkins/          # Jenkinsfile CI/CD pipeline definition
├── docker/           # Isolated Dockerfiles for each service
└── README.md
```

---

## 🛠️ Tech Stack

| Layer       | Technology                          |
|-------------|-------------------------------------|
| Frontend    | React 18, Tailwind CSS, Vite        |
| Backend     | Java 17, Spring Boot 3, Maven       |
| Database    | PostgreSQL                          |
| CI/CD       | Jenkins                             |
| Container   | Docker, Docker Compose              |
| Orchestration | Kubernetes (kubectl + manifests)  |
| Version Control | Git / GitHub                    |
